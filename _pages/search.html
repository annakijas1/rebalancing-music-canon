This is a simple search system. It will match most non-grammatical words. It is not case-sensitive, however you must include diacritics (e.g. Gr√∏ndahl) if they appear in your search term. Not all results will be highlighted on the page.

Type in your search below:

You can search for text that appears in the citation for a composition, such as a title, composer's name, movement title, publisher, or year, or that corresponds to the incipit, including title, instrument, ensemble type, movement information.

To search by ensemble type, use the following metadata terms to find incipits. Descriptions for each ensemble type are available in the project wiki.
  • solo
  • chamber-vocal
  • chamber-instrument
  • choral
  • band
  • orchestra

To search according to instrumentation indentified in the composition, use the following terms. You can also search by instrument name (e.g. flute, piano, etc.).
  • woodwinds
  • brass
  • electronics
  • strings
  • keyboard
  • percussion
  • pitched percussion
  • voice
